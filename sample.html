{:class="container": { :class="jumbotron bg-success": {h3: class="text-primary": HTON} } {:class="well": {h4:: Description} {p:: {strong::This} page is encoded in HyperText Object Notation.
Each object is converted into the HTML tag specified by the hton.js script at the end of this file.
If no tag is specified, then the object defaults to a div tag. } {p:: Each HyperText Object has {strong::three} sections which correlate to the parts of an HTML tag: {ul:: {li:: The first section is where the tag is specified. } {li:: The second is where the tag's attributes are enumerated.} {li:: The last section is where the tag's contents are enclosed.} } These three sections are separated by : colons.
Each HyperText Object is required to contain at least 2 colons, irrespective of whether a section is empty or not.
Colons in the attribute strings and in the last section will be ignored. } {h4::Notes} {ul:: {li:: If desired, raw HTML can be directly included in the last section. } {li:: All HyperText Objects will receive a closing tag that matches the opening tag.} {li:: In the last section, each occurrence of double brackets instead of single like {{{{ or }}}} will not be converted to HTML but will instead be escaped with the double brackets becoming single brackets. Remember to separate adjacent closing brackets with a space of other character so that they are not interpreted as getting escaped. } } {h4:: Implementation} {p:: The hton.js script can be included anywhere in the page, and does not run until the window has finished loading. } {p:: To get the script to convert the notation to html, simply place it in an html tag with the class="HTON". } } }
